  2. **Goals** — Official contract for: structure quality, provenance quality, OCR confidence / page status, retry history; consumers (review, preview, future estimate panel) read **structured** scores, not a single opaque number.

For retry history, state that today only **phase progress** exists on `ParseProgressRecord` and event arrays may default empty until persistence exists.

  7. **Relationship to `mappingQuality.ts`** — Short paragraph: badges remain the current UX tier API; `parseScore` **subsumes** richer dimensions; `getMappingQualityTier` stays the compatibility surface (Phase `18-02` may add parallel derivation only).
