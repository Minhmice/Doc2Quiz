  <action>
Implement **`src/app/api/develop/mock/[slug]/route.ts`**:
- Parse `slug` from `context.params`.
- If **`!isAllowedDevelopMockSlug(slug)`** → `NextResponse.json({ error: "Not found" }, { status: 404 })`.
- If **`process.env.NODE_ENV !== "development"`** AND **`process.env.ALLOW_DEVELOP_MOCKS !== "1"`** → **404** JSON (same shape).
- If **`slug` contains `".."` or `/` or `\\`** (or equals `path.sep`) → **404** JSON before any filesystem access.
- Else `readFile` from **`path.join(process.cwd(), "example", slug, "code.html")`** (use `fs/promises`); on `ENOENT` return 404.
- Success: `new NextResponse(buf, { status: 200, headers: { "Content-Type": "text/html; charset=utf-8", "Cache-Control": "no-store" } })`.
  </action>

    <automated>npm run build</automated>
  </verify>

    - `rg -n '\\.\\.' "src/app/api/develop/mock/"` exits 0 (explicit traversal guard in source).
