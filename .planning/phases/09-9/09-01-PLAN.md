  <action>
Implement `export type MathSegment = { kind: "text" | "math"; value: string; display?: boolean }` and `export function splitMathSegments(input: string): MathSegment[]`.

**Minimum rules (document in file header):**
1. Scan for `$$ ... $$` non-greedy **first**; inner becomes `{ kind: "math", display: true }`.
2. Then scan remainder for `$ ... $` non-greedy for `{ kind: "math", display: false }`.
3. Unpaired `$` remains **text** (do not throw).

Keep implementation **pure** (no DOM). Repo has **no** Vitest today — **do not** add a test runner in this wave. Instead, export a **`__DEV__`‑gated** `assertSplitExamples()` function at bottom of the same file (only called when imported with side effect in dev — **optional**) OR document **3 worked examples** in `09-01-SUMMARY.md` with input → segment JSON (executor must paste real outputs).

**Minimum bar:** `splitMathSegments` must pass manual sanity in SUMMARY for: (1) plain text no dollars, (2) `$x^2$`, (3) `$$\frac12$$` with surrounding text.
